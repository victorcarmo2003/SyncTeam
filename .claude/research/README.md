# Pesquisas

Resultados de pesquisa de API/documentação, salvos pelo agente `researcher`.

Formato: um arquivo por pergunta, `YYYY-MM-DD-<slug>.md`, contendo pergunta,
resposta objetiva, detalhes/ressalvas, fontes com URL e data de acesso, e nível
de confiança (alta = doc oficial; média = fórum consistente; baixa = indício).

Antes de codar sobre uma API nova, os agentes de código consultam esta pasta;
se a resposta não estiver aqui, a tarefa volta para o `researcher`.
